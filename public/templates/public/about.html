Sobre o Caça Fantasma

Bem-vindo ao Caça Fantasma!

Este projeto foi desenvolvido para ajudar a comunidade da UFRRJ a localizar os ônibus internos, carinhosamente apelidados de "Fantasminhas".

Como funciona:

  • Usuários podem optar por compartilhar anonimamente sua localização enquanto estão no Fantasminha.
  • O sistema agrupa essas localizações para estimar a posição atual dos ônibus.
  • Todos podem visualizar no mapa a localização estimada dos Fantasminhas ativos.

Este é um projeto colaborativo e visa facilitar o dia a dia dos estudantes, professores e técnicos da Rural.

Tecnologias utilizadas:

  • Python com Django para o backend.
  • HTML, CSS e JavaScript para o frontend.
  • Leaflet.js para a exibição do mapa interativo.

Contribua, compartilhe sua localização e ajude a manter o sistema preciso!
